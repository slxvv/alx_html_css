# CSS Advanced

This project styles a webpage for SmileSchool using advanced CSS techniques, based on a Figma design. It includes a header, banner, quotes, videos, membership, FAQ, and footer sections, built with HTML and CSS only. The CSS is defined in `styles.css` and linked to `index.html`.

## Files
- `index.html`: Main HTML file
- `styles.css`: CSS styles
- `images/`: Folder for images (not included in this repository)

## How to Run
1. Ensure all images referenced in `index.html` are placed in the `images/` folder.
2. Open `index.html` in a web browser.

## Requirements
- No external libraries (except Font Awesome for icons).
- W3C-compliant HTML/CSS.
- Uses Source Sans Pro font (via Google Fonts) and Spin Cycle OT (if available locally).

## Resources
- [Learn to Code HTML & CSS](https://learn.shayhowe.com/html-css/)
- [MDN Web Docs](https://developer.mozilla.org/)
- [Figma Design](https://www.figma.com/) (access via provided link)
- Fonts: Source Sans Pro, Spin Cycle OT

   ```bash

   ```

   ```bash
   cd alx_html_css/css_advanced
   ```
